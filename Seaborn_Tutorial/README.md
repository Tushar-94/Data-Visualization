## ðŸ“Š What We'll Learn

### ðŸ”· Line Charts
- **sns.lineplot**
   - Best for showing trends over time.
   - Multiple lines can compare patterns across different groups.

### ðŸŸ© Bar Charts & Heatmaps
- **sns.barplot**
   - Ideal for comparing quantities across categories.
- **sns.heatmap**
   - Reveals color-coded patterns in numeric tables.

### ðŸ”µ Scatter Plots
- **sns.scatterplot**
   - Shows the relationship between two numeric variables.
   - Add color to represent a third categorical variable.
- **sns.regplot**
   - Adds a regression line to help visualize linear relationships.
- **sns.lmplot**
   - Draws multiple regression lines for color-coded groups.
- **sns.swarmplot**
A categorical scatterplot showing how a numeric variable varies across categories.

### ðŸ“ˆ Distribution Plots
Understand how values in your dataset are distributed.

- **sns.histplot**
   - A histogram showing the frequency distribution of a numeric variable.
- **sns.kdeplot**
   - Smooth, estimated distribution curves (1D or 2D).
- **sns.jointplot**
   - Combines a 2D KDE plot with individual KDE/histograms for deeper insight.
